About Carithm AI

Carithm AI is a free, AI-powered car diagnostic tool designed to help users identify vehicle problems quickly and accurately. You can describe symptoms in plain language or enter OBD-II codes to receive mechanic-level insights—no technical knowledge required.

🔧 How It Was Built

  • Custom-built symptom and fault code databases
  • Condition-based rule matching for common and complex problems
  • Natural language mappings for real-world phrasing like “car shakes” or “won’t start”
  • Gradio and open-source tools for a fast, responsive interface
  • AI integration via Together.ai’s API for context-based suggestions

The goal is to make car diagnostics less stressful, more reliable, and completely free—by putting the power of AI in the hands of everyday drivers.
